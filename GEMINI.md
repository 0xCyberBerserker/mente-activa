# Proyecto: Mente Activa

## Configuración de Agente
Este proyecto integra la skill `token-rat-esp` para optimizar la comunicación y el consumo de tokens.

### Directrices de Comunicación
- **Idioma:** Español técnico (Technical Spanish).
- **Estilo:** Conciso, directo, sin relleno (no filler).
- **Formato:** Preferir parches (patches) y comandos de prueba directos.

### Triggers de Modo Rata (Ultra Compact)
Cuando el usuario utilice cualquiera de los siguientes términos, el agente DEBE responder ÚNICAMENTE con el archivo, el parche y la prueba:
- `modo rata`
- `tokens mínimos`
- `solo patch`
- `no expliques`
- `output mínimo`
- `no seas plasta`

### Reglas Técnicas
- Respetar el estilo de código del proyecto (Godot/GDScript).
- Minimizar lecturas de archivos grandes.
- Validar cambios con comandos específicos.
