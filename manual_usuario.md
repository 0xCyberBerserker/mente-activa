# User Manual

## English

Mente Activa is a Godot 4 educational app for short memory, language, and attention activities.

It is not a medical product, not a diagnostic tool, and not a treatment.

## Intended Use

- Short, calm activities.
- Large buttons and clear visual hierarchy.
- No time pressure.
- Friendly feedback.
- Optional support from family members or caregivers.

## Current Activities

- Daily pairs.
- Complete proverb.

## Getting Releases

Use GitHub Releases or the public GitHub Pages index linked from the repository README.

Current target platforms:

- Linux `x86_64`
- Linux `ARM64`
- Windows `x86_64`
- Android `arm64-v8a`

## Notes

- Commercial or profit-oriented use requires a prior commercial license.
- See `docs/README.md` for release, license, and contributor documentation.
