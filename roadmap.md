# Roadmap

## English

## Current State

- Alpha `0.0.7`.
- Godot 4.6.3 project.
- Manual release workflows.
- Target platforms: Linux `x86_64`, Linux `ARM64`, Windows `x86_64`, Android `arm64-v8a`.
- Public release index through GitHub Pages.

## Next

- Keep activity data validation strict and easy to run.
- Keep accessibility principles visible in new activities.
- Keep release artifacts, checksums, manifest, and public site coherent.
- Preserve the addon security-audit gate.

## Later

- Add more activities only when they match the short, calm, accessible design principles.
- Expand platform validation after release artifacts are proven on real devices.

# Roadmap

## Español

## Estado actual

- Alpha `0.0.7`.
- Proyecto Godot 4.6.3.
- Workflows de release manuales.
- Plataformas objetivo: Linux `x86_64`, Linux `ARM64`, Windows `x86_64`, Android `arm64-v8a`.
- Índice público de releases mediante GitHub Pages.

## Próximo

- Mantener estricta y sencilla la validación de datos de actividades.
- Mantener visibles los principios de accesibilidad en nuevas actividades.
- Mantener coherentes artefactos de release, checksums, manifest y web pública.
- Preservar el gate de auditoría de seguridad de addons.

## Más adelante

- Añadir más actividades sólo cuando encajen con los principios de diseño corto, tranquilo y accesible.
- Ampliar validación por plataforma después de probar artefactos en dispositivos reales.
